# Guion comercial sugerido para el demo PLI

## Apertura

PLI no es un mapa. Es una herramienta para decidir dónde invertir, dónde corregir y dónde no abrir.

## Narrativa del demo

1. Primero mostramos la red actual y los candidatos.
2. Después convertimos cada ubicación en score, ROI y riesgo.
3. Luego aislamos canibalización para evitar ventas ficticias.
4. Finalmente armamos un portafolio con business case y memo ejecutivo.

## Pregunta para el cliente

¿Cuánto les cuesta hoy una mala apertura, una tienda saturada o una ubicación que vende bien pero le roba venta a otra tienda?

## Cierre

La propuesta de BDS es construir el POSAR, entrenar el modelo, integrarlo al comité de expansión y crear una disciplina de aprendizaje post-apertura.
